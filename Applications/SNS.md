h1. Description 

Amazon Simple Notification Service (SNS) is a cloud service for coordinating the delivery of push messages from software applications to subscribing endpoints and clients. All messages published to Amazon SNS are warehoused across several availability zones to prevent loss.

h1. Main Features 
 
General :  
* highly scalable, flexible and cost-effective capability to publish messages from an application and immediately deliver them to subscribers or other applications. 
* Can group multiple recipients using topics. 
* Topic is an access point for allowing recipients to dynamically subscribe for identical copies of the same notification. 
* Instantaneous push based delivery 
* Simple API and easy integration with application 
* Flexible message delivery 
* Inexpensive 
* Web-based interface 

Push Notification to : 
* Apple 
* Google 
* Fire OS 
* Windows devices 
* Android devices in China
