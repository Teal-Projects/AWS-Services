h1. Description 

AWS provides the service AWS Secrets Manager for easier management of secrets. Secrets can be database credentials, passwords, third-party API keys, and even arbitrary text. 

h1. Main Features 
 

h5. General : 
* Similar to systems Manager Parameter Store.
* Charge per secret stored and per 10'000 API calls.
* Automatically rotate secrets.
* Apply the new key/password in RDS for you
* Cross-account access.
* Generate random secrets.
 

h1. Cost Strategies 

Pay per secret stored and per 10'000 Api calls.
