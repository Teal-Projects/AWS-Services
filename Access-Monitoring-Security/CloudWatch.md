h1. Description 

AWS CloudWatch is a suite of tools that encompasses a wide range of cloud resources, including collecting logs and metrics; monitoring; visualization and alerting; and automated action in response to operational health changes.

h1. Main Features 
 

h5. General : 
* You can create cloudwatch alarm  
* You can create dashboard. 
* Create alarm to be notified when a threshold is hit. 
* Create Events, respond to state changes in AWS resources. 

h5. Can Monitor : 
* Compute 
** EC2 Instances 
*** By default events are monitored every 5minutes you can set it up for 1 minutes. 
** Autoscaling Groups 
** Elastic Load Balancers 
** Route53 Health Checks 
* Storage & Content Delivery 
** EBS Volumes 
** Storage Gateways 
** CloudFront 

Cloudwatch & EC2 level Metrics : 
* CPU 
* Network 
* Status Chack 
* Underlying hypervisor whether it's running or not 
* Underlying EC2 instances
